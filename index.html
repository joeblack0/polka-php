Bienvenido

Fácil, rápido, seguro, gratis, en español...

Polka-php es un marco de desarrollo echo en y para php. Permite desarrollar aplicaciones mucho más rápido ya que no es necesario empezar desde cero. Polka-php brinda varias herramientas para el desarrollo de aplicaciones, cuenta con poderosas librerías, ayudantes y varios componentes que hará más placentero escribir código, dejando que programador se concentre en la lógica de su aplicación y no en la estructura. Polka-php brinda una estructura básica pero poderosa sobre la cual se puede desarrollar aplicaciones simples como complejas.

Características

  • Modelo Vista Controlador (mvc)
  • Modelo de Abstracción de datos propio
  • Manejador de plantillas propio
  • Manejos de sesiones
  • Url limpias (amigables)
  • Manejos de formularios
  • Manejos de seguridad
  • Disparadores, hachas...
  • Librerías, Ayudantes propias y privadas
  • y muchas otras...

Soporte para base de datos

  • Mysql 3, 4 y 5
  • PostgreSQL
  • Oracle CAll Interface
  • SQLite 2 y 3
  • ODBC v3
  • Microsoft SQL Server / SQL Azure
  • otros

Es para ti si quieres

  • Pequeña curva de aprendizaje
  • Poca (casi nada) configuración
  • Un marco de desarrollo pequeño y poderoso
  • Excelente rendimiento
  • Amplia compatibilidad con los alojamientos estándar

  • Cero código/s intrusivo/s
  • Soluciones simples a sistemas grandes y/o complejos
  • Pasar más tiempo lejos de la computadora
